# Marvel E-Commerce Full-Stack Project

This is a comprehensive e-commerce platform dedicated to Marvel merchandise, featuring a React frontend and a Spring Boot backend.

## 🚀 Project Structure
- Project-Marvel-Frontend: Developed using React (Vite). Handles the user interface, product catalog, and shopping experience.
- **Project-Marvel-Backend**: Developed using Java Spring Boot. Manages the API, business logic, and database connectivity.
- **DataBase**: Contains the MySQL script and setup instructions to initialize the store data.

## 🛠️ Tech Stack
- **Frontend:** React.js, Vite, CSS3
- **Backend:** Java, Spring Boot, Maven, Spring Data JPA
- **Database:** MySQL Workbench
- **Tools:** VS Code, Spring Tool Suite (STS)

## 📦 Features
- Product categories (Clothes, Action Figures, Keychains, etc.)
- Order management system
- Responsive UI design
- Real-time database integration

## ⚙️ Quick Setup
1. **Database:** Import the `.sql` file in the `DataBase` folder using the instructions provided there.
2. **Backend:** Open the backend folder in STS, update `application.properties` with your MySQL credentials, and run as a Spring Boot App.
3. **Frontend:** Navigate to the frontend folder, run `npm install` and then `npm run dev`.
